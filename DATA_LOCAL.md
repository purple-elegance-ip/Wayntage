# Local Data Storage

The large datasets used in this project are stored locally to keep the repository size manageable and avoid GitHub's file size limits.

## Excluded Directories
- `data/`: Contains processed meeting and CAD data.
- `CCAD Data/`: Contains raw data downloads and scraper outputs.

## How to get the data
If you need access to these files, please contact the project maintainer or run the ingestion scripts located in the `scripts/` directory to regenerate them.
